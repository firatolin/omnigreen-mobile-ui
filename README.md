# OmniGreen Mobile UI

Mobile UI/UX implementation for **OmniGreen**, an AI-powered agriculture application for Ethiopian farmers.

## Features

- FAYDA ID onboarding UI
- OTP verification flow
- AI Assistant UI (Text & Voice)
- Disease detection camera UI
- Loan & fertilizer request screens
- Multilingual-ready UI (Amharic, Afaan Oromo, English)

## Tech Stack

- Figma
- Adobe Illustrator
